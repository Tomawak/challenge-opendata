Challenge OpenData

Ce projet a pour but de permettre de visualiser plusieurs jeux de données de la manière la plus compréhensible pour le lecteur.

Les jeux de données que notre équipe a sélectionnés concernent l'Assemblée Nationale française.

Nous avons voulu représenter les similarités entre députés de partis différents ou non.
